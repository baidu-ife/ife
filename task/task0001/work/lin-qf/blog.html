4.12 CSS

遇到的问题和解决方法

。。。

首先是index.html中间的那三个div的高度一致问题，如果内容确定的话那么直接设定高度就行了但是在内容不确定的情况下，就不能固定高度，我用了flex布局的方法，那么就可以是三个div的高度和包含它们的div的高度一致，并且可以根据内容自动调整高度。
然后是窗口缩小时，header的导航部分会有部分没有背景色，主要有两种解决方法，一是设max-width为980px,而是直接再为header加一层背景。
然后是about.html中的那个图片定位及border的问题，刚开始我是让图片根据最外层容器定位的，但是这样的话图片跟内容就没有办法联系起来，于是我将整个容器分成四个模块，每个模块分成左右两边，每个图片都根据左边的来定位。然后就是教育经历那个年份和文字结合起来的，不能把年份单独定位，所以将显示年份的span和信息放在一个容器里一起定位
还有就是border颜色的问题，因为之前一直根据alpha值来调整颜色的深浅，但是这样做的话border的颜色会跟背景颜色重合起来，所以还是用rgb()来调整颜色。
最后是那个瀑布流问题，我刚开始用的都是flex，但是这样没办法将第一列再分成两边，于是我在第一列又用了浮动布局。
