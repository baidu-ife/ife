img标签遇到的问题

容器内的img标签与容器之间存在2px的空隙

问题：容器中的img标签和容器底部之间会有2px的间距

参考：[2px White Area Underneath Of Images] (http://stackoverflow.com/questions/21153123/strange-2px-white-area-underneath-of-images-possible-css-error)

解决方法： 

    #gallery > div > img {
        display:block;
    }
    或者：
    #gallery > div > img {
        vertical-align:top;
    }

inline-block间距问题

并排inline-block元素之间存在空隙

问题：在同一个容器中的并排inline-block元素之间，存在空白间距，和margin,padding无关。

参考：[Fighting the Space Between Inline Block Elements] (https://css-tricks.com/fighting-the-space-between-inline-block-elements/)

原因：inline-block标签之间存在空白字符，在页面上就会表现为空白间隙

解决方法：

1、使用注释把元素之间的空白处注释掉，或者在要使用inline-block的标签之间不留空格或者换行

2、使用负的margin，大概4px的margin可以消除空白

        nav a {
          display: inline-block;
          margin-right: -4px;
        }

3、把inline-block的父元素的font-size设置为0，在其子元素中再还原font-size数值：

        nav {
          font-size: 0;
        }
        nav a {
          font-size: 16px;
        }

并排等高div实现

使用table-cell实现并排div高度高度撑满容器

问题：实现容器中并排等高div并且自适应文字内容高度

解决方法：使用table-cell实现

兼容性见[CSS Table display](http://caniuse.com/#search=table-cell)

table-cell使用例子：

[这里](http://jsfiddle.net/FhQwb/1/)和[这里](http://jsfiddle.net/fm54T/)

实现：

        .table{
            display:table;
            border-collapse:separate;
            margin:0 -10px 0 -10px;
            border-spacing:10px;
        }
        .cell {
            display:table-cell;
        }

table-cell元素不受margin的影响，详见[Why is a div with “display: table-cell;” not affected by margin?] (http://stackoverflow.com/questions/16398823/why-is-a-div-with-display-table-cell-not-affected-by-margin)

要移除第一列左边和最后一列右边的`border-spacing`，把容器的`margin-left`和`margin-right`属性设置为负值

其中的border-spacing定义了并排div的间距，margin用来消除第一个和最后一个元素的左右边距。

参考：

[CSS table border spacing inside only](http://jsfiddle.net/marcstober/60dnac4a/1/)
