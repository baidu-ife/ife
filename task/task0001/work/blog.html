4.12 CSS

The First

收到任务的第一天

看到任务的第一天并没有着急写代码，而是把链接的资料看了看，BFC这个概念之前也只是知道有这么个东西，看过概念单并不明白，于是除了链接的那篇讲BFC和haslayout的文章外，又查了些资料，不说完全理解了BFC也解决了之前的一些疑惑，之后写盒子的时候遇到问题也有意识的想一想是否和BFC有关。

关于浮动，之前面试的时候被问到过为什么要清楚浮动，当时只答了防止父级塌陷，现在结合BFC，两厢验证，又有了更深理解。

4.12 CSS

Thee Second

马马虎虎不起眼

做这次的任务的心情可以说是跌宕起伏，由于时间比较短，我只有三天用来做，其中又有一些不得不做的杂事，老师大创要的东西，还有正常的课程，作业，当然，能逃的差不多都逃了，室友一条短信点名了，就从工作室一路狂跑过去，恩……锻炼了身体……嘻嘻，这不是重点。

时间的仓促我写的就越来越急还很粗糙，写完了两个页面自己都不想看了，又想着写都写完，再有时间就重构整理一下，然后这个想法也没实现……唉，汗颜。遇到了两个问题让我烦恼了好久结果是乌龙，但是还是要记住的。

第一个问题是锚链接问题，我写完之后link,visited都不能用，结果同学一语惊醒梦中人，我的链接都是同一个#，所以所以……

第二个问题是我好几处都用了css3 的nth-of-type()选择器，做完之后考虑兼容IE8不支持，时间又不多了就只能使用class了.匆忙添加完class之后发现样式不对了，在回头看了css才发现，太着急没注意优先级权重，再次更改完之后还是不对。我就心情烦躁了，站起身接个水转了一圈，回来再看的时候一眼就找出了错误。.class1.class2之间没有空格啊啊啊啊啊……

都是一些很小的问题但是有时候不细心就没注意到~~还是很重要的

4.12 CSS

The Three

总结

这次做这个任务的收获分为两部分，一方面是技术，一方面是思想。个人觉得第二个方面的收获更大。

技术方面

一是学习了BFC和haslayout

二是学习了透明度的渐变。

三是学习了css3的多栏布局。（IE8不支持）

思想方面

一是因为不允许使用js，写这次的页面的时候有种回到我刚开始学习前端的时候但是又不一样了，很多东西习惯性的用js。例如瀑布流布局，时间轴效果，tab选项卡。

二是思路的变化，做选项卡的时候鼠标移到a的时候要去掉border-bottom，我就想不要js怎么做嘛，然后就想到了可以把border-bottom的颜色变成白色……
