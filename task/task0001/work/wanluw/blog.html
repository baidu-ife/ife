github 中总结了很多的资源，为了扎实基础，便从头开始看，系统的回顾了 html 和 css 。

通过阅读资料，对于 css 中的定位有了更深刻的了解，对应 task0001 中的四张设计稿， position 和 float 的大量应用，也使得我的 css 布局定位更加得心应手。

同时对盒模型的理解也更加透彻，对 width 和 height 的设定也更加明确。

设计稿中 height 与 line-height 的设定， padding 的处理，表单和表格的使用，这些都使我的 html 和 css 进步了很多。
