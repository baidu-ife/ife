HTML的基本介绍

--简单介绍一下html里的基本标签

position 属性 通过使用 position 属性，我们可以选择 4 种不同类型的定位，这会影响元素框生成的方式。 position 属性值的含义： static 元素框正常生成。块级元素生成一个矩形框，作为文档流的一部分，行内元素则会创建一个或多个行框，置于其父元素中。 relative 元素框偏移某个距离。元素仍保持其未定位前的形状，它原本所占的空间仍保留。 absolute 元素框从文档流完全删除，并相对于其包含块定位。包含块可能是文档中的另一个元素或者是初始包含块。元素原先在正常文档流中所占的空间会关闭，就好像元素原来不存在一样。元素定位后生成一个块级框，而不论原来它在正常流中生成何种类型的框。 fixed 元素框的表现类似于将 position 设置为 absolute，不过其包含块是视窗本身。

PHP的基本介绍

--简单介绍一下PHP的用法

1、php的预定义常量 _FILE_ php程序文件名 _LINE_ php程序行数 E_ERROR 这个常量指到最近的错误处
2、define（）定义常量函数 constant（）动态获取常量 defined（）判断一个常量是否已经定义 一旦定义就不能改变或取消 get_defined_constants()获取当前已定义的所有常量 bool define(string name,mixed value[,bool case_insensitive]) string name 常量名字 mixed value 常量的值 bool case_insensitive 可选参数 指定大小写敏感 true为不敏感
3、strtotime() 将时间转换为UNIX时间戳 int strtotime(string time[,int now]);如果参数TIME的格式是绝对时间，则NOW不起作用；如果参数TIME的格式是相对的，其对应时间有NOW提供 ceil()返回不小于VALUE的下一个整数 float ceil(float value);
4、<<向左移位 >>向右移位 ~按位取反
5、date(string format,int timestamp); format 指定日期和时间输出的格式 timestamp 可选参数，是定时间戳，如果没有时间戳可用time（）
